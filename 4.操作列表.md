## 4.4 使用列表的一部分

### 4.4.1 切片
    players = ['charles', 'martina', 'michael', 'florence', 'eli']
    # 方式一
    print(players[0:3])  # 打印索引为0，1，2的元素
    print(players[:3])   # 打印索引为0，1，2的元素
    # 方式二
    print(players[3:5])  # 打印索引为3，4的元素
    print(players[3:])  # 打印索引为3，4的元素
    # 方式三
    print(players[:])  # 打印全部
    print(players)  # 打印全部
    
### 4.4.2 遍历切片
